usage: lime.py [-h] [--all] [--cert] [--algorithm] [--ciphers] [--protocols] [--headers] [--compression] [--options] url

Check website security.

positional arguments:
  url            URL to check

options:
  -h, --help     show this help message and exit
  --all          Run all checks
  --cert         Check SSL certificate
  --algorithm    Check SSL certificate signature algorithm
  --ciphers      Check weak SSL/TLS ciphers
  --protocols    Check SSL/TLS protocols
  --headers      Check HTTP headers
  --compression  Check compression settings
  --options      Check allowed HTTP methods
